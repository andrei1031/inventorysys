# Inventory System - Feature List

## ✅ Completed Features

### Core Features

- Add/Edit/Delete inventory items
- Multi-Inventory Support (create/switch between inventories)
- Search and sort functionality
- Low stock indicators (<10 shows red, <25 shows orange)
- CSV Export per inventory
- Cloud sync with Supabase

### Activity & History

- Activity Log - Track all changes (add, edit, delete, export)
- Item History - Full transaction history per item

### UI/UX

- Dark Mode - Toggle between light and dark themes
- Low Stock Alerts - Visual banner for low stock items
- Print View - Clean printable version (Ctrl+P)

### Archive System

- Archive functionality - move items to archive
- Restore from archive

### Keyboard Shortcuts

- Ctrl+N - Focus new item input
- Ctrl+F - Focus search
- Ctrl+D - Toggle dark mode
- Escape - Close modals

## Notes

- Uses Supabase for cloud storage
- Data persists across sessions
- Excluded user login/signup as requested
